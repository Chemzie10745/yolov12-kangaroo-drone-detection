# YOLO Detection Models - Setup and Usage Guide

This project contains YOLO object detection models for drone and kangaroo detection, including both standard and augmented models for comparison.

## 🚀 Initial Setup
### : Environment Setup
Unzip the drone file first and from the google drive link unzip the kangaroo file as well, make sure to read each readme.md file in both drone and kangaroo folders 

Ensure you have Anaconda/Miniconda installed with all dependencies in the `yolov12` environment.

## 🔍 Running Detections

### Prerequisites

1. Open **Anaconda Prompt**
2. Activate the environment:
   ```bash
   conda activate yolov12
   ```
